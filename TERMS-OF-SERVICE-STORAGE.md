## 1. STORAGE LIMITS

### 1.1 Tier-Based Limits
Your file upload limits are determined by your subscription tier:

| Tier | Files Per Wedding | Storage Per Wedding | File Size Limit |
|------|------------------|---------------------|-----------------|
| Free | 25 files | 250MB | 50MB per file |
| Premium | 50 files | 500MB | 50MB per file |
| Featured | 100 files | 1GB | 50MB per file |
| Elite | 200 files | 2GB | 50MB per file |

### 1.2 Limit Calculation
- Limits apply **per wedding/booking**, not total across all weddings
- Only the latest version of each file counts toward your limit
- Deleted files do not count toward your limit
- Archived weddings (90+ days after wedding date) are automatically deleted

### 1.3 Exceeding Limits
If you attempt to upload files beyond your tier limits:
- Upload will be **blocked** with clear error message
- You will be prompted to upgrade to a higher tier
- No additional charges will be incurred without explicit upgrade

## APPENDIX: TECHNICAL SPECIFICATIONS

### Supported File Types
- **Documents:** PDF, DOC, DOCX, TXT
- **Images:** JPG, JPEG, PNG, GIF, WEBP
- **Spreadsheets:** XLS, XLSX, CSV
- **Presentations:** PPT, PPTX
- **Archives:** ZIP (for multiple files)

### Recommended Best Practices
1. Compress images to reduce file size
2. Use PDF format for contracts (more secure)
3. Name files descriptively (e.g., "Johnson_Wedding_Contract_2024.pdf")
4. Delete old versions after uploading new ones
5. Review files quarterly and delete unnecessary ones
